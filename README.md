# C# Console (Compiled)



## VS Code

`dotnet run one two three`

## Developer Command Prompt (Visual Studio)

`dotnet run one two three`

Open Developer Command Prompt for VS2015 (or some other version e.g. VS2017)

### Compile

`csc Program.cs`

`./Program.exe one two "three 3"`

add quotation marks for spaces
